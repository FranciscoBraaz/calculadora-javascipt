## Projeto Calculadora


Esse projeto é uma modesta extensão da atividade prática desenvolvida no curso de JavaScript da Hcode. Algumas pequenas alterações foram feitas para colocar os conhecimentos adquiridos em prática, por exemplo:

- Redesign da calculadora em HTML e CSS;
- Correções de bugs presentes na atividade original;
- Algumas implementações um pouco diferentes do projeto original.

O projeto foi desenvolvido em HTML, CSS e JavaScript.
